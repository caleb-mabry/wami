wami - Where Am I?

Why wami?

Built for developers who work across multiple projects

Zero Config

Works instantly in any repo. No setup files, no configuration needed. Just run wami.

Multi-Language

Supports Node.js (npm, yarn, pnpm, bun), Python (poetry, uv, pipenv, pip), and more coming soon.

Command History

Remembers commands you run with arguments. Never retype complex commands again.

Interactive Editing

Add arguments on-the-fly, edit commands before running, fix typos instantly.

Smart Search

Instantly filter through 50+ scripts. Search by name, command, or description.

Mono-repo Ready

Intelligently detects workspaces and shows context. Always finds the nearest package.
